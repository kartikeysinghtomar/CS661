Make sure that you have python installed.
Make sure you have vtk installed in your system.
Replace the Isabel_3D.vti in " reader.SetFileName('Isabel_3D.vti') " with the file path from your system.

while running the code you'll be prompted with 'Do you want to use the Phong Shading filter? yes or no'.
choose among them for the desired output.
at last, you'll see a message that the image has been generated, and then a window with the required image will get generated.
