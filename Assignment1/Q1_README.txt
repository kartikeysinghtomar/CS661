Make sure to have Python, vtk and numpy installed.
Replace the Isabel_2D.vti in " reader.SetFileName("Isabel_2D.vti") " with the path of file in your system.

When running the script, you'll be asked to input the Isovalue which you want to see in your plot. you'll be prompted with "what isovalue do you want to take?? " specify any isovalue between (-1438, 630) (any floating point number) 
(range mentioned in the assignment) and a file named isocontour.vtp will
get generated in the path you'll put. You can load and view that file on Paraview .

At the end, you'll see a message that "File has been generated" and the program will end.
When viewing the file in paraview, make sure to change the colour of the background to something other than white because the colour of the isocontour will be white. Thank you.
